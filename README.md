# japan-mapcode

## Development Environment

### Setup Requirements

- Install [Pnpm](https://pnpm.io/installation)

### Dev mode

```bash
pnpm i
pnpm dev
```

Open `chrome://extensions/`, enable `Developer mode`. Click `Load unpacked` button and selct `dist` folder.

### Build

```bash
pnpm build
```
